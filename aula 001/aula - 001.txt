------ variaveis ------

* nome simbolo que você vai dar para um tipo de valor
* para declarar variavel precisa usar a palavra "var"


exemplo dele:
var x
(ele criou a varivel x, mas nao deu nenhum valor para ela)


e se só colocoar x?
-> volta "undefined" porque nao tem nenhum valor 


* para atribuir usamos o =  (x = 0) e assim não é mais undefined


posso mudar?
-> sim, x = 1 (e assim explica o nome variavel, ela tem um valor que pode mudar/variar)


* Js também tem outros tipos de valores, que eu posso atribuir
                    x = 1.2
1.2 é um tipo de numero, mas que é conhecido como FLOAT/REAL, e na matematica é conhecido como numero decimal


* quando eu chamo a varivel, ela retorna um valor
 
 exemplo dele:
> var nomequalquer = 10

> nomequalquer 
10



------ tipos de dados ------

*js aceita varios tipos de dados

° number 
° string (sequencia de caracteres dentro de '' ou "")
° boolean (true ou false)
° null (quando a variavel nao tem nenhum valor)
° undefined (quando nao existe valor)
° object {} (um objeto tem propriedade e métodos)
propriedade 
    ⬇️
tudo que o objeto tem 
    ⬇️
exemplo dele: 
> var pessoa = {}
> pessoa = {
    altura: 1.87,
    peso: 90
}
> pessoa
{altura:1.87, peso: 90}

como faço para pegar um dado especifico?
-> usa o ponto

>pessoa.altura
1.87


da para mudar o valor do objeto?
-> sim, dá pra sobrescrever
> pessoa.altura = 1.90


da para acessar um objeto assim?
-> sim 
> pessoa["altura"]
1.9 
(mas o ponto é mais utilizado)



° array [] 
> var numeros = []
(criou um array vazio)

>numeros 
[]


como atribuir valor ao array?
> numeros = [1, 2, 3, 4]
> numeros 
[1, 2, 3, 4]


como faço para acessar o array?
o array é acessado por indíces (começando do 0)

para ver o array selecionado:
>numeros[0]
1
